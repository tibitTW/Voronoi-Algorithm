線段

在尋找 hyperplane 的過程中時常需要尋找一段中垂線段被建構時使用的兩點，因此我同時記錄了原始的兩點以及中垂線段；線段的紀錄方式則記錄起點及終點，若一條線段為開放性線段則記錄與邊界之交叉點。 


class BisectionLine:
    def __init__(self, line, p1, p2, fill=None):
        self.line = line # (x1, y1, x2, y2)
        self.p1 = p1 # (p1x, p1y)
        self.p2 = p2 # (p2x, p2y)

Voronoi Diagram

我設計了一個 class 用於紀錄VD圖的點及線段，為了方便 convex hull 尋找，此 class 中另外紀錄了組合 convex hull 的所有點；而兩個 VD 圖在 merge 時需要改變的線段僅為一個 VD 圖中的開放性線段，因此亦將線段依照是否為開放性線段紀錄在不同的 list 中 


class VD:
    def __init__(self, points=None, lines=None, CH_points=None, CH_lines=None):
        self.points = [] # all points
        self.CH_points = [] # convex hull points
        self.lines = [] # cutted lines
        self.CH_lines = [] # infinite rays

Hyperplane

由於 hyperplane 為連續點依序連直線所組成，因此在尋找過程只需要依序紀錄每個點座標即可，直到找到完整的 hyperplane 之後再轉為各個線段


hyperplane = [(x1, y1), (x2, y2), (x3, y3), ..., (xi, yi)]
